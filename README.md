\# LLM Compliance Assistant



A Retrieval-Augmented Generation (RAG) application that answers compliance and policy-related questions using PDF documents.



\## Features



\* PDF document ingestion

\* Text chunking using LangChain

\* Semantic embeddings using Sentence Transformers

\* Vector search with FAISS

\* Context-aware answer generation using Google Gemini

\* Environment-based API key management



\## Architecture



PDF Documents

→ Chunking

→ Embeddings

→ FAISS Vector Store

→ Context Retrieval

→ Gemini LLM

→ Final Answer



\## Technologies



\* Python

\* LangChain Text Splitters

\* Sentence Transformers

\* FAISS

\* Google Gemini

\* python-dotenv



\## Example Question



How long should customer records be retained?



\## Example Answer



Customer records must be retained for 7 years.
